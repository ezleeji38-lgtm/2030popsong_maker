## 검수 포인트 (사용자 확인용)

- 중복 의심 행: 5번(편의점 새벽)과 14번(편의점 콜라)이 비슷한 톤 — 5는 썸/탐색, 14는 사귐/기념의 시점으로 구분
- 비 4회 (3, 6, 8, 20) / 눈 2회 (10, 13) / 봄 4 / 여름 4 / 가을 3 / 겨울 5 / 환절기 1 — 봄·여름·겨울 균형
- 모든 영어 제목은 Suno에 그대로 들어가도 무난한 길이 (3 단어 이하)
- 한국어 내용은 챗봇이 가사 변환할 때 "내용 = E열" 으로 주입할 수 있는 한 문장 단위

## 다음 단계

1. 사용자가 컨셉 검수 → 빼고 싶거나 더할 행 표시

5. 각 행에 F/G(원곡·원가사) 수동 입력 → J(새 가사) Claude/Gemini 자동 변환 → Suno 호출 → L(Music URL) 채움
