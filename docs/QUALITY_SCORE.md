# Song Maker — Quality Score

## 1. 품질 측정 기준

각 Stage의 출력물 품질을 관리하는 기준.
구체적인 자동 검증 항목과 통과/실패 판정은 [VERIFICATION.md](VERIFICATION.md)의 Gate 1~6에 정의되어 있다.
이 문서는 자동 검증 이외의 품질 기준을 다룬다.

---

## 2. 자동 검증 (Gate)

Gate 1~6에서 자동 수행되는 검증. 상세는 `VERIFICATION.md` 참조.

| Gate | 자동 검증 요약 |
|------|---------------|
| Gate 1 | API 응답, 결과 수, 데이터 유효성 |
| Gate 2 | 필수 입력 완료, 곡 수 범위 |
| Gate 3 | 오디오 파일 존재 + 크기 + 무결성 |
| Gate 4 | 이미지 파일 존재 + 디코딩 + 해상도 |
| Gate 5 | 영상 스트림 + 크기 + 길이 |
| Gate 6 | 업로드 성공 + URL + 메타데이터 |

---

## 3. 사용자 판단 품질 기준

자동 검증으로 확인할 수 없어 사용자가 직접 판단하는 항목:

### Stage 3 — 곡

| 항목 | 판단 기준 |
|------|-----------|
| 곡 완성도 | 곡이 자연스럽게 들리는가 (어색한 끊김, 반복 없음) |
| 장르 적합성 | 입력한 장르/분위기가 실제 곡에 반영되었는가 |
| 가사 품질 | 가사가 주제와 관련 있고 자연스러운가 |

### Stage 4 — 이미지

| 항목 | 판단 기준 |
|------|-----------|
| 분위기 적합성 | 이미지가 곡의 분위기와 어울리는가 |
| 시각 품질 | 아티팩트, 왜곡, 부자연스러운 요소가 없는가 |
| 썸네일 가독성 | 썸네일 텍스트가 읽기 쉬운가 |

### Stage 5 — 영상

| 항목 | 판단 기준 |
|------|-----------|
| 자막 가독성 | 자막이 배경과 겹쳐 읽기 어렵지 않은가 |
| 페이드 자연스러움 | 시작/끝 전환이 자연스러운가 |

---

## 4. 재생성 기준

사용자 판단에서 불합격 시:
- Stage 3: `songmaker create` 재실행 (새 곡 생성)
- Stage 4: `songmaker image <song_id>` 재실행 (이미지만 재생성)
- Stage 5: `songmaker render <song_id>` + 옵션 변경 (자막, 페이드 조정)
