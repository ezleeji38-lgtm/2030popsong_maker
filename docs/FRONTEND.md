# Song Maker — CLI Interface Design

> 이 프로젝트는 CLI 도구이다. 프론트엔드 웹 UI는 없다.
> 이 문서는 CLI 인터페이스(터미널 UX) 설계를 다룬다.

---

## 1. CLI 프레임워크

- **Typer** — 타입 힌트 기반 CLI 빌더
- **Rich** — 테이블, 진행바, 컬러 출력

---

## 2. 명령어 체계

```bash
songmaker [command] [options]
```

### 메인 명령어

| 명령어 | 설명 |
|--------|------|
| `songmaker run` | 전체 파이프라인 (Stage 1~6) |
| `songmaker trend` | 트렌드 조사만 |
| `songmaker create` | 사용자 입력 + 곡 생성 |
| `songmaker import <mp3_path>` | MP3 수동 임포트 (Suno 대안) |
| `songmaker image <song_id>` | 이미지 생성만 |
| `songmaker render <song_id>` | 영상 렌더링만 |
| `songmaker upload <song_id>` | 유튜브 업로드만 |

### 관리 명령어

| 명령어 | 설명 |
|--------|------|
| `songmaker config` | API 키 설정 (대화형) |
| `songmaker config show` | 현재 설정 표시 (키 마스킹) |
| `songmaker list` | 생성된 곡 목록 |
| `songmaker status <song_id>` | 곡 상태 상세 |

### 글로벌 옵션

| 옵션 | 설명 |
|------|------|
| `--region` / `-r` | 트렌드 지역 (기본: KR) |
| `--count` / `-n` | 생성할 곡 수 (기본: 1) |
| `--verbose` / `-v` | 상세 로그 출력 |
| `--project` / `-p` | 프로젝트 이름 지정 |

## 4. 에러 표시

```
  [에러] YouTube API 키가 설정되지 않았습니다.
  songmaker config 명령으로 설정하세요.

  [에러] FFmpeg가 설치되지 않았습니다.
  설치: https://ffmpeg.org/download.html

  [경고] Suno 곡 생성 실패. 재시도 중... (2/3)
```
