# 2030 채널 곡 컨셉 — 월별·날씨별 컴필레이션

발행 주기: 주 2-3회 → 카테고리당 30선 = 약 10주 분량
컬럼 매핑: 영어 제목(C) / 한국어 제목(D) / 한국어 내용 한 줄(E)
톤: 시적 한 컷, 한 문장 안에 한 이미지. 형용사 최소화, 명사·동사 중심.

> **확장 계획**: 각 월(1-12) + 비 + 눈 + 환절기 + 새벽 등 14개 카테고리 × 30선 = 420곡 목표.
> 이번 파일: 비 / 6월. 다음 차수에 ❄️ 눈 / 🌸 4월 벚꽃 → 그 다음 ... 순으로 누적.

## 톤 가이드 (계속 유지)

- **명사 중심 + 직접 진술 허용**: "사랑해", "보고싶어" 같은 직접 진술 OK. 단, 매 곡마다 직접 진술만으로 채우지 말고 사물·시간·장소와 섞기.
- **한 컷 한 문장**: 영화 컷처럼 한 장면만. 줄거리 X.
- **숫자 활용**: "60초", "두 정거장", "1분", "5미터" — 구체성이 시적 거리감.
- **체언 종결**: 가능하면 명사 종결로 마무리 (`...의 일 분`, `...한 사람`).
- **중복 회피**: 같은 사물(우산, 카페) 반복돼도 각도 달리. (안쪽 / 색깔 / 손잡이 / 두 자루 / 같이 쓴 후드 등)
