# Stage 2 — 사용자 입력 상세 설계

## 1. 목적

트렌드 보고 결과를 바탕으로 사용자가 곡의 장르, 분위기, 주제 등을 결정한다. CLI 대화형 프롬프트로 입력을 수집하고 `SongRequest` 모델로 변환한다.

---

## 2. 입력 항목

| 순서 | 필드 | 필수 | 타입 | 기본값 | 설명 |
|------|------|------|------|--------|------|
| 1 | genre | O | str | - | 장르 선택 |
| 2 | mood | O | str | - | 분위기 입력 |
| 3 | theme | O | str | - | 곡 주제 |
| 4 | lyrics_keywords | X | list[str] | [] | 가사 키워드 (쉼표 구분) |
| 5 | reference_song | X | str | None | 참고곡 (트렌드 번호 또는 곡명) |
| 6 | count | O | int | 1 | 생성할 곡 수 |

---

## 3. 입력 흐름

### 장르 선택

트렌드 분석에서 추출된 장르를 선택지로 제시:

```
? 장르를 선택하세요:
  1. 발라드 (트렌드 40%)
  2. K-pop (트렌드 35%)
  3. 인디 (트렌드 25%)
  4. 직접 입력
> 1
```

- 트렌드 장르 목록 + "직접 입력" 옵션
- 직접 입력 시 자유 텍스트

### 분위기 입력

```
? 분위기를 입력하세요 (예: 밝은, 슬픈, 몽환적, 에너지틱):
> 몽환적
```

- 예시를 힌트로 표시
- 자유 텍스트 입력

### 주제 입력

```
? 곡의 주제를 입력하세요 (예: 봄날의 설렘, 이별 후 성장):
> 봄날의 설렘
```

### 가사 키워드 (선택)

```
? 가사에 포함할 키워드 (쉼표로 구분, Enter로 건너뛰기):
> 벚꽃, 바람, 설렘
```

- 빈 입력 시 건너뜀
- 쉼표로 분리하여 `list[str]`로 변환

### 참고곡 (선택)

```
? 참고곡 번호 또는 곡명 (Enter로 건너뛰기):
> 1
```

- 트렌드 목록 번호 입력 시 해당 곡 정보 참조
- 곡명 직접 입력 가능
- 빈 입력 시 건너뜀

### 곡 수

```
? 생성할 곡 수 (1~10, 기본: 1):
> 1
```

---

## 4. 입력 검증 (Gate 2)

입력 완료 즉시 검증:

```python
def validate_input(request: SongRequest) -> list[str]:
    errors = []
    if not request.genre.strip():
        errors.append("장르를 입력해주세요.")
    if not request.mood.strip():
        errors.append("분위기를 입력해주세요.")
    if not request.theme.strip():
        errors.append("주제를 입력해주세요.")
    if not (1 <= request.count <= 10):
        errors.append("곡 수는 1~10 사이로 입력해주세요.")
    return errors
```

검증 실패 시 해당 항목만 재입력 요청 (전체 재입력 아님).

---

## 5. 입력 확인

모든 입력 후 요약 표시 + 확인:

```
  입력 요약
┌──────────────┬──────────────────┐
│ 장르         │ 발라드           │
│ 분위기       │ 몽환적           │
│ 주제         │ 봄날의 설렘      │
│ 가사 키워드  │ 벚꽃, 바람, 설렘 │
│ 참고곡       │ Spring Breeze    │
│ 곡 수        │ 1                │
└──────────────┴──────────────────┘

? 이 설정으로 진행하시겠습니까? [Y/n]: y
```

- `n` 입력 시 처음부터 재입력
- `Y` 또는 Enter 시 Stage 3으로 진행

---

## 6. SongRequest 생성

검증 + 확인 통과 후 `SongRequest` 객체 생성:

```python
request = SongRequest(
    genre="발라드",
    mood="몽환적",
    theme="봄날의 설렘",
    lyrics_keywords=["벚꽃", "바람", "설렘"],
    reference_song="Spring Breeze",
    count=1
)
```

Gate 2 통과 시 → Stage 3 (곡 생성)으로 진행.
