## 3. OAuth 2.0 인증

### 최초 인증

```python
from google_auth_oauthlib.flow import InstalledAppFlow

SCOPES = [
    "https://www.googleapis.com/auth/youtube.upload",
    "https://www.googleapis.com/auth/youtube"
]

def authenticate(client_secret_path: Path, token_path: Path) -> Credentials:
    if token_path.exists():
        creds = Credentials.from_authorized_user_file(str(token_path), SCOPES)
        if creds.valid:
            return creds
        if creds.expired and creds.refresh_token:
            creds.refresh(Request())
            save_token(creds, token_path)
            return creds

    # 최초 인증: 브라우저 열림
    flow = InstalledAppFlow.from_client_secrets_file(str(client_secret_path), SCOPES)
    creds = flow.run_local_server(port=0)
    save_token(creds, token_path)
    return creds
```

## 4. 영상 업로드

### 업로드 함수

```python
def upload(
    creds: Credentials,
    video_path: Path,
    metadata: VideoMetadata,
    thumbnail_path: Path | None = None
) -> str:
    """영상 업로드 → YouTube URL 반환"""

    youtube = build("youtube", "v3", credentials=creds)

    body = {
        "snippet": {
            "title": metadata.title,
            "description": metadata.description,
            "tags": metadata.tags,
            "categoryId": "10",     # Music
        },
        "status": {
            "privacyStatus": metadata.privacy,  # "private"
            "selfDeclaredMadeForKids": False,
        }
    }

    media = MediaFileUpload(
        str(video_path),
        mimetype="video/mp4",
        resumable=True,
        chunksize=256 * 1024  # 256KB 청크
    )

    request = youtube.videos().insert(
        part="snippet,status",
        body=body,
        media_body=media
    )

    response = resumable_upload(request)
    video_id = response["id"]

    # 썸네일 설정
    if thumbnail_path and thumbnail_path.exists():
        set_thumbnail(youtube, video_id, thumbnail_path)

    return f"https://youtu.be/{video_id}"
```

### Resumable Upload

```python
def resumable_upload(request) -> dict:
    """청크 단위 업로드 + 진행 표시"""
    response = None
    with Progress() as progress:
        task = progress.add_task("YouTube 업로드 중...", total=100)
        while response is None:
            status, response = request.next_chunk()
            if status:
                percent = int(status.progress() * 100)
                progress.update(task, completed=percent)
    return response
```

### 썸네일 설정

> **전제 조건**: YouTube 계정이 전화번호 인증(youtube.com/verify)되어 있어야 한다.
> 미인증 시 썸네일 API 호출이 403으로 실패한다. Gate 6에서 비차단 경고로 처리.

```python
def set_thumbnail(youtube, video_id: str, thumbnail_path: Path) -> bool:
    """썸네일 설정. 실패 시 False 반환 (비차단)."""
    try:
        youtube.thumbnails().set(
            videoId=video_id,
            media_body=MediaFileUpload(str(thumbnail_path), mimetype="image/png")
        ).execute()
        return True
    except HttpError as e:
        if e.resp.status == 403:
            # 계정 미인증 또는 권한 부족
            console.print("[경고] 썸네일 설정 실패. YouTube 계정 인증을 확인하세요.")
            return False
        raise
```

def generate_title(song: Song) -> str:
    return f"{song.theme} | {song.genre}"
    # 예: "봄날의 설렘 | 발라드"

### 설명 생성

```python
def generate_description(song: Song) -> str:
    lines = [
        f"🎵 {song.theme}",
        f"",
        f"장르: {song.genre}",
        f"분위기: {song.mood}",
        f"",
    ]

    # 가사 포함
    if song.lyrics_path and song.lyrics_path.exists():
        lyrics = song.lyrics_path.read_text(encoding="utf-8")
        lines.append("--- 가사 ---")
        lines.append(lyrics)
        lines.append("")

    lines.append("Generated by Song Maker")
    return "\n".join(lines)
```

### 태그 생성

```python
def generate_tags(song: Song) -> list[str]:
    tags = [song.genre, song.mood, song.theme]
    tags.extend(song.lyrics_keywords)
    tags.extend(["AI music", "AI 음악", "Song Maker"])
    return tags[:30]  # YouTube 태그 제한
```

## 6. 공개 전환

업로드는 항상 비공개(private)로 실행. 사용자 확인 후 공개 전환:

```python
def set_public(creds: Credentials, video_id: str):
    youtube = build("youtube", "v3", credentials=creds)
    youtube.videos().update(
        part="status",
        body={
            "id": video_id,
            "status": {"privacyStatus": "public"}
        }
    ).execute()
```

CLI 흐름:
```
업로드 완료 (비공개): https://youtu.be/xxxxx

? 공개로 전환하시겠습니까? [y/N]: y
공개로 전환되었습니다.
```
